# Perceptron Algorithm on Iris Dataset

Implementation of the Perceptron learning algorithm for binary classification using the Iris dataset.

## Description
This project implements a perceptron from scratch to classify Iris flowers into two species: Setosa and Versicolor. The implementation includes:
- Data loading and preprocessing
- Perceptron training with loss tracking
- Visualization of decision boundaries
- Performance metrics over epochs

## Requirements
- Python 3.x
- NumPy
- Pandas
- Matplotlib

## Usage
```bash
pip install -r requirements.txt
python perceptron.py
